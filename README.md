# موقع التوعية بالتوحد 🌈

موقع شامل يساعد المجتمع على فهم مرض التوحد وكيفية التعامل مع الأشخاص المصابين به بطريقة سهلة وبسيطة.

## 📋 محتويات الموقع

- **صفحة رئيسية احترافية** - ترحيب وتعريف عام عن التوحد
- **5 خطوات بسيطة** - خطوات سهلة وواضحة للتعامل مع ذوي التوحد
- **نصائح مهمة** - إرشادات قيمة وعملية
- **صور توضيحية** - مقسم خاص بالصور والرسوم التوضيحية

## 🎯 الهدف

توعية المجتمع بخصائص التوحد وكيفية التعامل مع الأفراد المصابين به بطريقة احترامية وإيجابية.

## 🛠️ التكنولوجيا المستخدمة

- HTML5
- CSS3 (Responsive Design)
- تصميم متجاوب يعمل على جميع الأجهزة

## 📱 المميزات

✅ تصميم بسيط وسهل الاستخدام
✅ محتوى عربي واضح ومفيد
✅ خطوات عملية وقابلة للتطبيق
✅ ألوان مريحة للعين
✅ متوافق مع جميع الأجهزة (موبايل، تابلت، كمبيوتر)
